Project Structure

A detailed breakdown of the React component hierarchy and file organization.

src/
├── App.tsx // Main entry, global state manager
├── types.ts // Shared TypeScript interfaces
├── components/
│ ├── ProjectionView.tsx // t-SNE scatterplot (Canvas + D3 Zoom)
│ ├── ComparisonView.tsx // Comparison glyphs container
│ ├── RadarChart.tsx // SVG Path generator for star plots
│ ├── DominancePopup.tsx // Hover overlay for dominance details
│ ├── TabularView.tsx // Table + Diverging Bar Charts
│ └── QueryModifier.tsx // Filter control panel
└── services/
├── skylineService.ts // BNLS Algorithm (CPU)
├── webgpuService.ts // Compute Shaders (GPU)
└── dataService.ts // CSV Parser

Core Data Structures

Component API Reference

<ProjectionView />

UI Component Canvas Rendering

Renders high-dimensional data projected onto a 2D plane (using t-SNE or PCA coords). To maintain 60FPS with thousands of points, it uses an HTML5 <canvas> for rendering dots and D3.js only for handling Zoom/Pan events and calculating scales.

Props

Prop NameTypeDescription
dataDataPoint[]Full dataset with x/y coordinates.
skylineResultSkylineResultUsed to color points (Red = High Dominance).
selectedIdsstring[]IDs of currently pinned points (highlighted with stroke).
onSelect(id: string) => voidCallback when a point is clicked.
onHover(id: string | null) => voidCallback for hover tooltips.

<ComparisonView />

Complex Visualization D3 Force Layout

A dedicated view for comparing selected points. It uses a d3.forceSimulation to prevent overlap between the comparison glyphs. Each glyph is a composite visualization containing a Radar Chart and dominance pie charts.

Props

Prop NameTypeDescription
selectedIdsstring[]List of points to compare (max 4 recommended).
attributesAttributeConfig[]Dimensions to show on the radar axes.
skylineResultSkylineResultProvides dominance relationships between selected points.

Internal Logic

The component calculates Union Size and Exclusive Domination to draw the pie charts connecting the glyphs.

<RadarChart />

SVG Sub-Component

A pure functional component that generates the SVG path string for a star plot. Used internally by ComparisonView and DominancePopup.

Props

Prop NameTypeDescription
playerDataPointThe data object to visualize.
attributesAttributeConfig[]Defines the axes and their max values (for normalization).
sizenumberRadius of the chart in pixels.
showRankingbooleanIf true, draws ranking circles on the axes.

<DominancePopup />

Overlay

The detailed tooltip that appears when hovering over a dominance link. It visualizes why Point A dominates Point B by superimposing their radar charts. The "Better" point surrounds the "Worse" point in all relevant dimensions.

Props

Prop NameTypeDescription
skylinePointsDataPoint[]The dominating points (rendered with solid fill).
dominatedPointsDataPoint[]The dominated points (rendered with dashed lines).
x, ynumberScreen coordinates for positioning.

<TabularView />

Data Table Diverging Bar Chart

Lists all points with inline visualizations. Instead of raw numbers, it renders Diverging Bar Charts: Blue bars extend right (above average), Red bars extend left (below average).

It also computes Decisive Subspaces (combinations of 2 attributes where the point remains in the Skyline) when a row is expanded.

Props

Prop NameTypeDescription
dataDataPoint[]The list of points to display (supports filtering).
hoveredIdstring | nullUsed to highlight the corresponding row.
onSelectfunctionExpands the row to show the subspace matrix.

<QueryModifier />

Control Panel

Allows the user to set global filters (e.g., "Price < 500"). These filters update the filteredData state in App.tsx.

Props

Prop NameTypeDescription
filtersFilterConfig[]Current list of active filters.
onFiltersChangefunctionUpdates the filter state in the parent.
